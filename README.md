# Morgan count fingerprints 

## Model Identifiers
- Slug: morgan-counts
- Ersilia ID: eos5axz
- Tags: morgan,	ML,	descriptor

## Model Description
Morgan fingerprint done with radius 3 and 2048 dimensions 
- Input: SMILES 
- Output: Vectors 
- Model type: Regression
- Mode of training: Pretrained
- Experimentally validated: No
